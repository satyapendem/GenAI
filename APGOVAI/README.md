# Simple RAG Chat Application

A clean Retrieval-Augmented Generation (RAG) application using:

- LangChain
- ChromaDB
- Ollama
- Llama 3.1 8B
- FastAPI
- React

---

# Prerequisites

Install:

- Python 3.10+
- Node.js 18+
- Ollama

Pull required models:

```bash
ollama pull llama3.1:8b
ollama pull nomic-embed-text
